# bdz-ecommerce

Vite + React + TypeScript + Tailwind + Firebase starter for a single-product storefront.

Quick start:
1. copy `.env.example` to `.env` and fill Firebase keys
2. npm install
3. npm run dev

Deploy:
- Firebase Hosting: use `firebase init` and `firebase deploy`
- Vercel: connect repo and set VITE_ env vars
